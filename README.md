BP Voice - Голосовой ввод давления

Вводите данные тонометра голосом. Распознавание работает через встроенный браузерный Web Speech API — ничего не передаётся на сервер.



Как пользоваться

Откройте сайт

Нажмите «Разрешить микрофон» (один раз)

Нажмите на кнопку микрофона

Продиктуйте: «Сто двадцать на восемьдесят, пульс шестьдесят»

Проверьте распознанные значения и нажмите «Добавить»

Поддерживаемые браузеры

Google Chrome (рекомендуется)

Microsoft Edge

Яндекс Браузер

Safari (macOS/iOS)

Firefox — не поддерживает Web Speech API.



Технологии

HTML / CSS / JavaScript (без фреймворков)

Tailwind CSS — стили

Chart.js — графики

Web Speech API — голосовое распознавание

Font Awesome — иконки
